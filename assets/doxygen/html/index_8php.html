Detailed Description

Script for displaying the main website page. Includes components. Displays error message pop-up in case of error context parameter set in GET request.

'session_invalid_id' and 'wrong_admin_rights' errors occurence means that the user managed to access the interface elements that require proper user authorization (of the levels of 'authorized user' of 'admin') without one, and leads to the mandatory interrupt of the current session and logout of the user.
